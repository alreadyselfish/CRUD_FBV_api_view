This is a readme file. It contains all the endpoints that you can use to access the CRUD API.

GET localhost:8000/api/list/
This fetches the list of all the products

POST localhost:8000/api/add/
This can be used to add products.

PATCH loaclhost:8000/api/pk/
pk = primary key
can partially update product with primary key = pk

PUT loaclhost:8000/api/pk/
pk = primary key
can update product with primary key = pk

GET loaclhost:8000/api/pk/
pk = primary key
can lookup product with primary key = pk

DELETE loaclhost:8000/api/pk/
pk = primary key
can delete product with primary key = pk
